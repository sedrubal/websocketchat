Websocket chat
==============

A small and simple chat website using tornado and websockets.
This is only for playing around with websockets and not for production ;)

Usage
-----

 - Install tornado: `pip install tornado`
 - Run the app: `./main.py`
 - Go to: http://localhost:8888/

Features
--------

 - websockets
 - python3
 - tornado
 - Bootstrap
 - Javascript
